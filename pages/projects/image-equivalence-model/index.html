    Image Equivalence Model

    A convolutional neural network for determining if two modified images originate from the same source.

    This project is intended to illustrate the process of creating a convolutional neural network for the purpose of computer vision and image processing. To this end, I derived a new dataset from the CIFAR100 dataset, creating image pairs that were modified by various torchvision transforms. Half of the pairs contained images derived from the same source image, and half of the pairs contained images derived from different source images. The model created for this project takes an image pair and determines whether the images come from the same source or not.
